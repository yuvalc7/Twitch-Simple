## About the App 
 a clone of web application Twitch TV   
 
 ### logged in vs not logged in  
   logged in :  
   - create unlimited streams  
   - edit and delete stream  
   
   not logged in :  
   - view a list of streams  
   - view video for a single stream      


### This App using  
- google Oauth  
- lodash  
- redux  
- portal  
- react-router  
- JSON-Server
